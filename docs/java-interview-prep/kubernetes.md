# Kubernetes Interview Questions

This section covers questions related to Kubernetes (K8s), an open-source system for automating the deployment, scaling, and management of containerized applications.

### Topics:
- Kubernetes Architecture (Control Plane, Worker Nodes, etcd)
- Core Components (Pods, Services, Deployments, ReplicaSets)
- ConfigMaps and Secrets
- Networking in Kubernetes (Services, Ingress)
- Persistent Storage (PersistentVolumes, PersistentVolumeClaims)
- Helm for package management
- `kubectl` command-line tool
- Namespaces and Resource Quotas
- Liveness and Readiness Probes
