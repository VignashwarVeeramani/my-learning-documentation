# Docker Interview Questions

This section covers questions related to Docker, a platform for developing, shipping, and running applications in containers.

### Topics:
- Containers vs. Virtual Machines
- Dockerfile and its key instructions (`FROM`, `RUN`, `COPY`, `CMD`, `ENTRYPOINT`)
- Docker Images and Layers
- Docker Containers and Lifecycle
- Docker Compose for multi-container applications
- Docker Networking (Bridge, Host, Overlay)
- Docker Volumes and Data Persistence
- Docker Hub and Image Registries
- Basic Docker CLI commands
