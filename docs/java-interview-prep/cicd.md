# CI/CD Interview Questions

This section covers questions related to Continuous Integration (CI) and Continuous Delivery/Deployment (CD), which are practices that automate the software development and release process.

### Topics:
- What is CI/CD?
- Key Benefits (Faster Releases, Improved Quality)
- CI/CD Pipeline Stages (Build, Test, Deploy)
- Popular Tools (Jenkins, GitLab CI, GitHub Actions)
- Blue-Green Deployment
- Canary Deployment
- Infrastructure as Code (IaC) with tools like Terraform or Ansible
- Pipeline as Code
- Difference between Continuous Delivery and Continuous Deployment
