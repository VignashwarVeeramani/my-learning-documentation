# Spring Boot Interview Questions

This section covers questions related to the Spring Boot framework, which simplifies the development of stand-alone, production-grade Spring-based applications.

### Topics:
- Spring Boot Basics (Starters, Auto-Configuration, `@SpringBootApplication`)
- Spring MVC (`@RestController`, `@RequestMapping`, `ResponseEntity`)
- Dependency Injection and Inversion of Control (IoC)
- Spring Data JPA and Repositories
- Spring Security
- Actuator and Health Checks
- Externalized Configuration (`application.properties`, YAML)
- Profiles
- Testing with Spring Boot
