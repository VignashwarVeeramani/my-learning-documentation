# AWS Interview Questions

This section covers questions related to Amazon Web Services (AWS), focusing on the core services commonly used in cloud-native application development.

### Topics:
- **Compute:** EC2, Lambda, ECS, EKS
- **Storage:** S3, EBS, EFS
- **Database:** RDS, DynamoDB
- **Networking:** VPC, Route 53, API Gateway, Load Balancers (ALB, NLB)
- **Security & Identity:** IAM (Users, Roles, Policies)
- **Messaging:** SQS, SNS
- **Monitoring:** CloudWatch
- Infrastructure as Code (IaC) with CloudFormation
