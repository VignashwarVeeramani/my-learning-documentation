# Microservices Interview Questions

This section covers questions related to the microservices architectural style, focusing on the design, principles, and patterns used to build distributed systems.

### Topics:
- Microservices vs. Monolithic Architecture
- Key Principles (Single Responsibility, Loose Coupling, High Cohesion)
- Service Discovery (Eureka, Consul)
- API Gateway (Spring Cloud Gateway, Zuul)
- Fault Tolerance and Resilience (Hystrix, Resilience4j)
- Inter-service Communication (REST, gRPC, Message Queues)
- Distributed Tracing (Zipkin, Sleuth)
- Configuration Management (Spring Cloud Config)
- Design Patterns (Saga, CQRS, Event Sourcing, Strangler)
- CAP Theorem
